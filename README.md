# Yury Portfolio V2

A dark, modern portfolio for Yury Li — Marketing, Sales & AI Workflow Specialist.

## Stack

- Vite
- React (JavaScript)
- Tailwind CSS
- Motion
- GitHub Pages

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm preview
pnpm lint
```

## Content updates

Edit `src/data/portfolio.js` for navigation, copy, skills, projects, workflow steps, and contact details. Each page section is an independent component under `src/components`.

## Deployment

The Vite base path is `/REACTWEB/`. Push the `main` branch to `Logic-io/REACTWEB`, then set **Settings → Pages → Source** to **GitHub Actions**. Future pushes to `main` deploy automatically.

Temporary site URL: `https://logic-io.github.io/REACTWEB/`
